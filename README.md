## 🧠 Sobre este repositorio

Este repositorio reúne una colección de proyectos desarrollados durante el curso **Funval 2026**, enfocados en el crecimiento progresivo dentro del desarrollo frontend.

Aquí no solo encontrarás resultados finales, sino también el proceso, los errores, las mejoras y la evolución como desarrollador.

---

## 🎯 Objetivo

Este proyecto tiene un propósito **100% educativo** 📚

* Documentar el aprendizaje paso a paso
* Reforzar conceptos clave del frontend
* Servir como referencia para otros estudiantes
* Construir una base sólida para proyectos reales
